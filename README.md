# Earthquake
This app displays a list of recent earthquakes in the world.<br>
It uses the USGS Earthquake API available at: https://earthquake.usgs.gov/fdsnws/event/1/

## Support
1- Udacity<br>
2-Android Developers:https://developer.android.com
